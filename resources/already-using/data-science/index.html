Data Science with General Bots

Talk to Your Data

General Bots turns data access from a specialist task into a conversational interaction. The Talk to Data feature lets business users ask questions in natural language and receive answers drawn directly from databases, spreadsheets, and documents — no SQL, no Python, no dashboard configuration required. Under the hood, the system translates natural language queries into structured data operations, executes them, and presents results in readable format. Data scientists configure the data sources and define the semantic layer; business users interact through conversation.

  • Natural language queries against databases and spreadsheets
  • No SQL or Python required for business users
  • Semantic layer configured by data scientists, used by everyone
  • Results presented in readable conversational format

Vector Retrieval and RAG

Retrieval-Augmented Generation is the backbone of accurate, grounded AI responses. General Bots includes built-in vector storage and retrieval that indexes your documents, web pages, and knowledge bases. When a user asks a question, the system retrieves the most relevant passages and feeds them to the LLM as context, producing answers that cite source material rather than inventing information. The vector retrieval pipeline supports chunking strategies, embedding model selection, and similarity thresholds — all configurable without code changes.

  • Built-in vector storage and retrieval pipeline
  • RAG architecture grounds LLM responses in source material
  • Configurable chunking, embedding models, and similarity thresholds
  • No-code configuration for data science workflows

BASIC-Powered Data Workflows

For data operations that go beyond simple retrieval, BASIC scripts provide a programmable layer between the user and the data. A BASIC tool can execute database queries, call external APIs, transform results, apply business rules, and format the output — all within a single, readable script. This approach replaces the typical data pipeline stack of notebooks, schedulers, and microservices with one file that any team member can read, modify, and deploy. Data scientists write the logic once; the bot executes it on demand, at scale, across any channel.

  • BASIC tools execute queries, transform data, and apply business rules
  • Single readable script replaces notebooks, schedulers, and microservices
  • On-demand execution at scale across any channel
  • Any team member can read, modify, and deploy data workflows

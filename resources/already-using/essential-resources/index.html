Essential Resources

Documentation and References

The General Bots documentation covers everything from initial setup to advanced orchestration. The documentation site is the primary reference — covering BASIC syntax, LLM tool definitions, channel configuration, and deployment patterns. API references document every REST endpoint for programmatic bot management. The admin guide covers user management, package deployment, and system configuration. All documentation is version-controlled alongside the codebase, so it stays synchronized with each release rather than drifting into obsolescence.

  • Documentation site: comprehensive BASIC, tools, and configuration reference
  • REST API documentation for programmatic bot management
  • Admin guide for user management and system configuration
  • Version-controlled documentation synchronized with releases

Operational Tools

Running bots in production requires operational visibility. General Bots includes built-in tools for monitoring bot health, inspecting conversation logs, and tracking LLM token consumption. The admin dashboard shows real-time metrics: active conversations, average response times, error rates, and channel-specific usage. Conversation logs are searchable and filterable, making it possible to diagnose issues, audit responses, and identify improvement opportunities. Token consumption tracking helps teams manage LLM costs before they become surprises.

  • Real-time dashboard: active conversations, response times, error rates
  • Searchable conversation logs for debugging and auditing
  • LLM token consumption tracking to manage costs proactively
  • Channel-specific usage metrics for capacity planning

Community Knowledge Base

Beyond official documentation, the General Bots community contributes templates, tutorials, and solution patterns. The template marketplace provides plug-and-play bot packages for common use cases — customer support, knowledge retrieval, appointment scheduling, and data analysis. Community tutorials walk through real-world deployments with step-by-step instructions. The Help Center aggregates frequently asked questions and known issues with resolutions. When you encounter a problem, chances are someone in the community has already solved it and shared the solution.

  • Template marketplace with plug-and-play bot packages
  • Community tutorials with step-by-step deployment instructions
  • Help Center with FAQs and known-issue resolutions
  • Community solutions for common implementation challenges

FAQ

Frequently Asked Questions

Everything you need to know about General Bots.

What is General Bots?
General Bots is an open source AI bot platform that lets you build, deploy, and manage intelligent conversational agents. It provides LLM orchestration, multi-channel deployment, and a unique BASIC programming model that lets you author bot logic directly in Microsoft Word.
Is it really free?
Yes. General Bots is licensed under the MIT License, which means you can use, modify, and distribute it freely — including for commercial purposes. There are no per-seat or per-bot fees. You only pay for infrastructure and LLM API usage if you use external providers.
What does BASIC programming mean?
General Bots uses a simplified BASIC dialect as its primary programming language. You can write bot logic in .bas files using a familiar, readable syntax — and even author them directly in Microsoft Word. This lowers the barrier to entry compared to Python or JavaScript-based frameworks.
Which LLM providers are supported?
General Bots supports OpenAI (GPT-4, GPT-3.5), Anthropic (Claude), Google (Gemini), and self-hosted models via local inference. The orchestration layer lets you route different tasks to different models based on cost, latency, and capability requirements.
Can I self-host?
Absolutely. General Bots is designed for self-hosting. You can deploy it on your own infrastructure — on-premises, cloud, or hybrid — for complete data sovereignty and compliance with LGPD, GDPR, and other data protection regulations.
What channels are supported?
General Bots supports WhatsApp, Microsoft Teams, Slack, Google Chat, Instagram, SMS, Email, Web Chat, Facebook Pages, and more. A single bot can be deployed across all channels simultaneously with unified conversation context.
How is this different from LangChain or Bot Framework?
LangChain is a framework for building LLM applications in Python/JavaScript — it requires developers. Bot Framework is Microsoft's channel-focused SDK. General Bots combines both: it provides LLM orchestration like LangChain and multi-channel deployment like Bot Framework, but with a no-code/low-code BASIC programming model that non-developers can use directly in Word.
Who are the certified hosting providers?
Certified hosting providers include Pragmatismo, OERLabs, Map Agency, and Sentient. Dario Executive Sales provides Microsoft licensing and sales support. These providers offer managed General Bots deployments with guaranteed uptime, support, and compliance.

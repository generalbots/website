' enrollment.bas — Customer Bot
SUB enrollment()
TALK "Welcome! Let me help you."
HEAR name AS string
HEAR email AS string
SAVE customer TO database
TALK "Done, " + name + "!"
END SUB

FAQ

Frequently Asked Questions

Everything you need to know about General Bots.

What is General Bots?
General Bots is an open source AI platform that combines LLM-powered bots with LLM-assisted human attendants. Bots handle routine conversations autonomously across WhatsApp, Teams, Instagram, Telegram, Slack, and Web. When conversations need a human touch, the bot hands off seamlessly — and the LLM keeps working as a copilot, providing real-time tips, smart replies, sentiment analysis, and message polish for your agents. Built on Rust + Tokio for millions of concurrent connections.
Is it really free?
Yes. General Bots is licensed under the MIT License, which means you can use, modify, and distribute it freely — including for commercial purposes. There are no per-seat or per-bot fees. You only pay for infrastructure and LLM API usage if you use external providers.
What does BASIC programming mean?
General Bots uses a simplified BASIC dialect as its primary programming language. You can write bot logic in .bas files using a familiar, readable syntax — and even author them directly in Microsoft Word. This lowers the barrier to entry compared to Python or JavaScript-based frameworks.
Which LLM providers are supported?
General Bots supports OpenAI (GPT-4o, GPT-4, GPT-3.5), Anthropic (Claude), Google (Gemini), Groq, and self-hosted local models. A single PROMPT.md per project defines bot personality. The orchestration layer routes tasks to different models based on cost and capability — with semantic caching delivering a 95% hit ratio and NVIDIA CUDA acceleration when available.
Can I self-host?
Absolutely. Three deployment options: (1) Self-hosted on your own infrastructure with full control, (2) Managed hosting via Pragmatismo at YourCompany.pragmatismo.com.br, or (3) Hybrid with local runtime and cloud backup. Export everything as ZIP anytime — no vendor lock-in. All options include Argon2 hashing, JWT tokens, TLS 1.3, and built-in GDPR/LGPD/HIPAA compliance.
What channels are supported?
General Bots supports WhatsApp, Microsoft Teams, Slack, Google Chat, Instagram, SMS, Email, Web Chat, Facebook Pages, and more. A single bot can be deployed across all channels simultaneously with unified conversation context.
How is this different from Zendesk, Dialogflow, or Botpress?
Unlike Zendesk/Intercom (per-seat pricing, no AI-first), Dialogflow (no native WhatsApp, no human handoff, no CRM), or Botpress (no LLM orchestration, no enterprise features), General Bots is a complete platform: AI bots + human attendant copilot + built-in CRM + omnichannel — all self-hosted. Versus n8n/Zapier, it's conversational AI, not just workflows. Versus custom development, it's 90% faster deployment on a proven Rust architecture.
Who are the certified hosting providers?
Certified hosting providers include Pragmatismo, OERLabs, Map Agency, and Sentient. Dario Executive Sales provides Microsoft licensing and sales support. These providers offer managed General Bots deployments with guaranteed uptime, support, and compliance.

Core Platform

The foundation that powers everything. Authentication, storage, AI, and orchestration.

Server

The heart of the platform. Web server, WebSocket messaging, REST API, and request routing. Serves both the API and the web interface.

Auth / Identity

OAuth 2.0, OpenID Connect, JWT validation, and session management. Role-based access control (RBAC) for every endpoint. SSO-ready.

Shared / Database

Database models, common types, and shared utilities across all applications. PostgreSQL with managed schema definitions across all modules.

AI Engine

LLM provider orchestration. OpenAI, Anthropic, Groq, Azure, and local models. Automatic failover, token counting, streaming, and cost tracking across providers.

Drive / Storage

S3-compatible object storage. File upload, download, versioning, and automatic indexing for search and RAG. Works with MinIO, AWS S3, Wasabi.

Dashboards / Analytics

Real-time monitoring dashboards with system health metrics, business KPIs, and custom visualizations. Exportable and embeddable.

Meet / Video

Video conferencing with screen sharing, recording, transcription, and meeting notes auto-generation.

Documents, Sheets & Slides

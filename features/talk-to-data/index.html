Natural Language Queries

Ask Questions, Get Answers

Users interact with data using everyday language. General Bots translates questions into queries and returns clear, actionable answers.

  • No SQL Knowledge Needed

    Business users ask questions like “What were sales last quarter?” and get instant results. The LLM translates natural language into optimized SQL with joins, aggregations, and filters — automatically.

  • Contextual Follow-ups

    The bot remembers conversation context, enabling natural follow-up questions like “Break that down by region” or “Show me just Q4” without restating the original query.

  • Sub-Millisecond Queries

    Built on PostgreSQL with Diesel ORM, the database layer delivers sub-millisecond query performance. Complex analytical queries return results fast enough for real-time conversation.

  • Tables, Charts, or Natural Language

    Answers adapt to the question: simple numbers get a sentence, comparisons get a table, trends get a chart. The LLM chooses the best format for clarity.

Multi-Source Data

Connect to Any Data Source

Query across SQL databases, spreadsheets, REST APIs, and documents from a single conversational interface.

  • SQL & NoSQL Databases

    Connect to PostgreSQL, SQL Server, MySQL, MongoDB, and more with pre-built connectors. Automatic schema discovery means the bot understands your tables and relationships without manual configuration.

  • Spreadsheets & Files

    Import Excel, CSV, and Google Sheets data for instant conversational analysis. The ingestion pipeline handles formatting, headers, and data types — no manual data cleaning required.

  • Row-Level Security

    Role-based permissions ensure users only see data they're authorized to access. Sales reps see their pipeline; managers see the full team. Enforced at the database layer, not just the UI.

  • Audit Trail

    Every query is logged with the user, the natural language question, the generated SQL, and the results returned. Full compliance visibility for regulated industries.

Smart & Secure

Intelligent Data Interaction

AI-powered query generation with built-in security guardrails ensures accurate, safe access to your data.

  • AI Query Generation

    LLMs translate natural language into optimized SQL, handling joins, aggregations, and complex filters automatically.

  • Access Control

    Role-based permissions ensure users only query data they are authorized to see, with row-level security enforcement.

  • Result Visualization

    Answers can be presented as tables, charts, or natural language summaries depending on the query context.

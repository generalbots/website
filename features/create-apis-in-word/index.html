How It Works

Here's how it works: you write a BASIC script in a Word document, defining the logic step by step. SAVE a customer record. GET data from a database. CALL an external API. The compiler transforms your document into a compiled AST file that runs as a REST endpoint. No IDE, no deployment pipeline, no waiting for the engineering team. If you can write a numbered list of instructions, you can create an API. This is especially powerful for legacy system integration — expose mainframe or SQL Server data as modern JSON APIs without touching the original system.

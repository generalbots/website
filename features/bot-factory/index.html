Running one bot is easy. Running fifty — each with its own personality, knowledge base, and channel configuration — is a different story. That's why we built the Bot Factory: a system for creating, deploying, and managing bots at scale, from a single control plane.

Atomic Template System

Define a single PROMPT.md and configuration template, then spawn thousands of isolated bot instances. Update the master template and all child bots receive the update instantly. No manual redeployment required.

True Multi-Tenancy

Complete data isolation between tenants. Each organization's customer data, configuration, and conversations are cryptographically separated on the same core infrastructure.

Scale Without Limits

Orchestrate hundreds of specialized bots — HR, IT, Sales, Support — under a single management plane. Each bot has its own personality, knowledge base, and channel configuration.

How It Works

The magic is in the template system. Define your bot's personality once in a master PROMPT.md, configure its tools and knowledge bases, then spawn as many instances as you need. When you update the master template, every child bot receives the change instantly. No SSH, no redeploys, no manual sync. Each bot still has its own isolated data and conversation history — tenants never see each other's information.

Related Features

Each bot in the factory can use LLM Tools for AI-powered responses, connect to Broadcast for outbound messaging, and integrate with APIs in BASIC for backend logic.
